# Exploratory Data Analysis Project

This is focused on using statistics and data visualization to turn raw data into useful insights. It includes exploratory data analysis, data cleaning, and interpretation of results through clear and relevant charts.

To test the app: https://huggingface.co/spaces/will-7s/eda_app

## Overview

As a data scientist, one of the main goals is to make data understandable and actionable.  
Here, we analyze datasets using descriptive statistics such as mean, median, standard deviation, and distribution patterns. We also choose the most appropriate chart for each type of variable:

- **Quantitative variables**: histogram, box plot, scatter plot
- **Qualitative variables**: bar chart, pie chart when the number of categories is limited

## Goals

We covered two main case studies:

### 1. Banking Data Analysis
We use a Kaggle dataset to investigate bank card fraud.  
The analysis includes:
- Univariate analysis
- Bivariate analysis
- Conclusions about features to distinguish fraudulent transactions

### 2. Marketing Data Analysis (in progress...)
We study customer segments to identify the most profitable groups based on their interactions with marketing campaigns.  
The analysis includes:
- Univariate analysis
- Bivariate analysis
- Correlation analysis
- Business-oriented conclusions

## Methodology

The workflow follows four main steps:

1. **Exploration**  
   Inspect the dataset structure, variables, missing values, duplicates, and data types.

2. **Cleaning**  
   Prepare the data by handling missing values (for statistic analysis), inconsistent entries, and duplicates.

3. **Analysis**  
   Use descriptive statistics, distributions, correlations, and visualizations to uncover trends.

4. **Presentation**  
   Summarize findings and communicate insights clearly.

## Key Questions

The analysis is guided by questions such as:
- What are the distributions of the variables?
- Which charts are the most relevant for each variable type?
- Are there relationships between variables?
- What patterns can be observed in fraud or customer behavior?
- Which segments are the most profitable?
